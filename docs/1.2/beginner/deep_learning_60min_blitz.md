* 在高层上理解 PyTorch 的 Tensor 库以及神经网络。

*本教程假设你对 numpy 有基本的了解*

**注意**

- 确保你安装了 [torch](https://github.com/pytorch/pytorch) 和 [torchvision](https://github.com/pytorch/vision) 包。
